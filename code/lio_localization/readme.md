global_localization_old.py是最终版本的全局地图匹配定位程序
订阅全局地图（/map）、激光扫描点云（/cloud_registered）、里程计（/Odometry）和初始位姿（/initialpose）；
将实时激光扫描点云与全局地图进行 ICP 配准，计算map到odom的变换矩阵；
发布配准后的局部地图（/submap）、当前扫描点云（/cur_scan_in_map）和map_to_odom变换关系，实现激光定位。

原版全局地图全量加载，内存占用极高，初始化慢，新版引入FAISS 索引（L2 距离索引），支持按半径快速查询局部地图
原版无缓存机制，每次启动都重新解析全局地图，新版实现地图 / 索引缓存（~/.lio_map_cache），首次加载后缓存到本地
原版单线程执行，初始化 / 配准阻塞主线程，新版采用多线程异步处理（地图解析、索引构建、子图发布）
原版全局地图全量参与 ICP 配准，计算量巨大，新版动态按半径查询局部地图（配准半径可动态调整）
